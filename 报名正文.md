## 1. 创意名称 + 创意介绍

**创意名称**：Xcomputer — AI 桌面自动化助手

**想解决什么问题**：日常办公中大量重复性电脑操作（打开应用、整理文件、填写表格、跨应用复制粘贴）耗费时间精力，非技术用户面对系统维护、批量处理等任务更是无从下手。

**为什么会想到做这个**：灵感来源于大模型 Agent 的成熟。既然 AI 能理解自然语言、能调用工具、能观察结果循环决策，为什么不让它直接操控我们的电脑？每次想完成一个多步骤任务，都要手动点击、输入、切换窗口，这些机械操作本应交给 AI。DeepSeek 等国产大模型的能力已经足够支撑复杂的工具调用循环，Windows-MCP 框架也提供了完整的桌面操控能力，两者结合就能实现"一句话操控电脑"的体验。

**大概是什么产品**：Electron 桌面应用（Windows），三栏 IDE 风格界面，集成 DeepSeek 双模型（V4-Flash 意图分类 + V4-Pro 任务执行）和 Windows-MCP 框架，用户输入自然语言指令，AI 自主规划并执行电脑操作。

## 2. 目标用户及痛点

**面向哪些用户**：
- 职场白领：每天处理大量文档、表格、邮件，重复操作多
- 非技术人员：不熟悉命令行和系统操作，遇到技术问题就卡住
- 开发者：需要批量文件操作、终端命令、项目搜索等辅助
- 老年人/残障人士：操作电脑不便，希望用语言指挥

**在什么场景下使用**：
- 办公自动化："打开 Word 把会议纪要整理成表格"
- 文件管理："把桌面截图按日期分类到文件夹"
- 系统维护："清理临时文件并查看磁盘空间"
- 跨应用协作："把浏览器表格复制到 Excel"

**当前痛点**：没有 Xcomputer，用户必须手动一步步操作，跨应用切换容易出错，批量任务只能逐个处理，系统操作对非技术用户门槛极高。现有自动化工具（如 AutoHotkey）需要编程能力，RPA 软件学习成本高且不够智能。

## 3. 价值与意义

**效率提升**：将"手动多步"变为"一句话一步"。原本 10 分钟的文件整理，10 秒指令完成。AI 7×24 不知疲倦，释放人力专注创造性工作。经测试，批量文件处理、跨应用操作等场景效率提升 10 倍以上。

**社会价值**：降低电脑使用门槛，让老人、残障人士、非技术用户都能通过自然语言完成复杂操作。无需学习命令行和快捷键，消除数字鸿沟，让技术真正服务于每一个人。这是"AI 普惠"在桌面场景的具体落地。

**技术栈**：Electron 31 + React 18 + TypeScript 5 + Tailwind CSS + DeepSeek AI（V4-Flash/V4-Pro）+ Windows-MCP + ReAct Agent

**核心特性**：
- 自然语言理解：DeepSeek-V4-Flash 毫秒级意图分类
- ReAct 智能规划：DeepSeek-V4-Pro 自主决策，最多 20 轮循环
- 21 个工具：19 个 MCP 桌面操控 + 本地 File 文件操作 + 本地 Terminal 后台终端
- 高危操作确认：删除/注册表/进程终止自动弹窗确认
- 项目自带 Python+uv：用户无需额外安装任何工具链
