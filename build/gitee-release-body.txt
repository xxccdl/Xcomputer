## 🚀 性能优化版本

本次更新让 AI 流式输出更丝滑，同时大幅降低内存和 CPU 占用。

### 渲染优化
- **Store 层流式热路径**：streamingMessage 分离，token 更新从 O(n) 降至 O(1)
- **React.memo 冻结历史消息**：流式时只重渲染最后一条消息
- **Markdown 解析节流**：useThrottledValue 限制到 12fps（80ms），CPU 削减 80%+
- **O(1) 步骤查表**：stepsByMessageId Map 替代 O(n×m) filter
- **阈值门控虚拟化**：超过 80 条消息自动启用虚拟滚动，DOM 减少 90%

### 资源占用优化
- **后台节流**：backgroundThrottling + 禁用拼写检查
- **窗口隐藏降频**：USB/网络轮询 10s→60s，调度检查 30s→120s
- **GPU 合成优化**：移除 sticky banner backdrop-blur 减少滚动重绘

### 预期改善
| 指标 | 优化前 | 优化后 | 改善 |
|------|--------|--------|------|
| 流式 token 更新 | O(n) | O(1) | ~100x |
| 流式时重渲染组件 | 全部消息 | 仅最后 1 条 | ~Nx |
| Markdown 解析频率 | 每 token | ~12fps | 80% ↓ |
| 长会话 DOM 节点 | 100+ | ~10 可见 | 90% ↓ |
| 后台 CPU 占用 | 10s 轮询 | 60-120s | 85% ↓ |
